# Event Radar

全球事件监控与分析系统

## 功能

- 📡 实时全球事件流
- 🗺️ 交互式地图可视化
- 📊 金融市场数据监控
- ⏱️ 多时间维度筛选

## 部署

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/你的用户名/event-radar)

## 本地运行

```bash
python3 -m http.server 8080
```

访问 http://localhost:8080

## 技术栈

- 纯 HTML/CSS/JavaScript
- Leaflet.js 地图库
- 响应式设计
